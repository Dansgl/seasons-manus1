# Baby Seasons

A premium circular-economy platform for quarterly luxury baby clothing subscriptions with fixed 5-item boxes, inventory state management, and customer dashboard.

## Features

- Quarterly subscription boxes
- - Luxury baby clothing
  - - Circular economy model
    - - Customer dashboard
      - - Inventory management
       
        - ## Tech Stack
       
        - - TypeScript
          - - React
            - - Supabase
              - - Stripe
                - - Sanity CMS
